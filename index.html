Guide d'utilisation de l'extension Voiranime

Bienvenue dans l'extension Voiranime ! Cette extension vous permet de télécharger des épisodes depuis Voiranime.

Comment utiliser l'extension

  1. Ouvrez l'extension via l'icône dans la barre d'outils de Chrome.
  2. Entrez l'URL de la série Voiranime que vous souhaitez télécharger.
  3. Choisissez l'option de téléchargement (tous les épisodes ou un seul épisode).
  4. Cliquez sur le bouton "Télécharger".
